معد الاختبارات

تبسيط إعداد الاختبارات الورقية ونماذج الإجابة للمعلمين والمعلمات

( اسئلة اختبارات مخصصة مع الكليشة - نماذج اجابة - كل المراحل الدراسية - جميع المواد الدراسية )

اضافة للمتصفح تعمل على موقع ( عين ) تتيح للمعلمين والمعلمات امكانية اعداد اسئلة اختبار ونماذج اجابة بطريقتين :

الطريقة الاولى / اعداد اسئلة اختبار سريع وشامل لكامل المنهج مع نماذج الاجابة ( وتشمل انواع الاسئلة : صح وخطأ - اختر الاجابة الصحيحة - اختر الاجابات الصحيحة - الترتيب -اكمل الفراغ ويتم توزيع الاسئلة في طريقة ( اختبار سريع ) لتشمل جميع المنهج الدراسي حسب مستويات الصعوبة ثم يتم تصدير الاسئلة ونماذج الاجابة بهيئة ملف (word قابل للتعديل)

الطريقة الثانية / اعداد اسئلة اختبار تفصيلي لكل درس او الدروس المختارة - ويتم اعداد نماذج الاجابة ايضا وتشمل انواع الاسئلة ( صح وخطأ - اختر الاجابة الصحيحة - اختر الاجابات الصحيحة - الترتيب - اكمل الفراغ ) ويتم اختبار الاسئلة حسب نوع السؤال ودرجة الصعوبة ثم يتم تصدير الاسئلة ونماذج الاجابة بهيئة ملف (word قابل للتعديل)
